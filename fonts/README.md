# Fonts

Sample fonts for testing and examples.

## Included Fonts

- **DejaVuSans.ttf** - Sans-serif font for general text
- **DejaVuSansMono.ttf** - Monospace font for code/fixed-width text
- **DejaVuSerif.ttf** - Serif font
